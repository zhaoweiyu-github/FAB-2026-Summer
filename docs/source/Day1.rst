Day 1
=====

Machine-learning foundations for regulatory genomics
----------------------------------------------------

Please read the paper `Predicting the sequence specificities of DNA- and
RNA-binding proteins by deep learning <https://doi.org/10.1038/nbt.3300>`_.
The goal is to understand how sequence features can be used to predict
transcription factor binding.

Please also review the following scikit-learn materials:

1. `Data preprocessing <https://scikit-learn.org/stable/modules/preprocessing.html>`_.
2. `Decision trees <https://scikit-learn.org/stable/modules/tree.html>`_.
3. `Classification metrics <https://scikit-learn.org/stable/modules/model_evaluation.html#classification-metrics>`_.

Practice
~~~~~~~~

Build several scikit-learn classifiers on a small classification dataset and
compare accuracy, precision, recall, F1, AUROC, and AUPRC.

Optional
~~~~~~~~

If you are interested, begin exploring the basics of deep learning with
``PyTorch`` and implement a small classifier for tabular or sequence data.
