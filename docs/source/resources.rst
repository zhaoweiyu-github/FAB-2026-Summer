Submission checklist
--------------------

Before submitting an assignment, make sure your repository or notebook includes:

* A clear environment setup section.
* A single command or notebook order that reproduces preprocessing, training,
  and evaluation.
* A metric table on the held-out test set.
* Notes about class imbalance and threshold selection.
* A short interpretation of the result, including one limitation.
